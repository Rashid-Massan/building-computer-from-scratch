<h1>💻 Building a Computer from First Principles (NAND2Tetris)</h1>
Welcome to my journey of building a fully functional, modern computer system from scratch. Based on the acclaimed Nand2Tetris (Elements of Computing Systems) curriculum, this project traces a complete vertical slice through computer science starting with simple electricity-like logic gates and ending with a high-level object-oriented programming language and an operating system.

The core philosophy of this project is to demystify the "magic" of computers by implementing every single layer of the stack myself.

<h2>🛠️ Tech Stack & Tools</h2>
<ul>
  <li>

  </li>
  <li>
Nand2Tetris Software Suite: Hardware Simulator, CPU Emulator, and VM Emulator.
  </li>
</ul>
